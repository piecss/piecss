    breakpoint Since 0.1 

    @mixin breakpoint($breakpoints, $media) { ... }

    Description

    Expects a valid breakpoint expression like (min-width: 30em)
    A collection of pre-defined breakpoints are defined in settings/breakpoints. Override and add breakpoints in your project's settings file.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $breakpoints

    A valid breakpoint

    		 String — none
    $media

    The media to which this breakpoint applies

    		 Map — none

    Example

     $small_handheld: "(max-width: #{to-unit(480px,$unit,$default-font-size)})";
 @include breakpoint($small_handheld);

    column-behavior Since 0.1 

    @mixin column-behavior($column-float: false, $negative-right-margin: false) { ... }

    Description

    Mixin column-behavior: prepares an element to behave as a column. Widths must be set separately with column-width.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $column-float

    If true, render columns as left floated block element. If false, render as inline-blocks.

    		 Bool false
    $negative-right-margin

    If set, applies a negative right margin to force inline-blocks to render without gaps. Only use this if you cannot minify your markup.

    		 Bool, Number false

    inline-block Since 0.1 

    @mixin inline-block($vertical-alignment: baseline) { ... }

    Description

    Provides a cross-browser method to implement display: inline-block;. This replaces Compass's mixin by the same name with a more accessible implementation.
    See for valid property values: http://www.w3schools.com/cssref/prposvertical-align.asp

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $vertical-alignment

    Set $inline-block-alignment to none or false to disable the output of a vertical-align property in the inline-block mixin. Or set it to a legal value for vertical-align to change the default.

    		 String baseline

    Example

     @include inline-block(baseline);
